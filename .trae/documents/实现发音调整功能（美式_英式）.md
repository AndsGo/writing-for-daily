# 发音调整功能实现方案

## 功能目标
允许用户选择美式/英式发音，并提供谷歌和微软的多种语音选项

## 实现步骤

### 1. 扩展 SpeechService
- 修改 `speak()` 方法，支持按语音名称选择语音
- 添加 `getVoicesByLang(lang)` 方法，按语言分组语音
- 添加 `getVoiceByName(name)` 方法，按名称查找语音

### 2. 创建语音设置 Store
- 新建 `src/stores/voice.ts`
- 管理用户语音偏好设置
- 支持的设置：
  - `selectedVoice`: 当前选中的语音名称
  - `rate`: 语速（0.1-10）
  - `pitch`: 音调（0-2）

### 3. 创建设置页面
- 新建 `src/views/Settings.vue`
- 功能：
  - 语音类型选择（美式/英式）
  - 语音列表下拉框（显示可用语音）
  - 语速滑块
  - 音调滑块
  - 试听按钮
  - 保存设置

### 4. 更新路由配置
- 在 `src/router/index.ts` 添加设置页面路由
- 在导航栏添加"设置"菜单项

### 5. 更新播放功能
- 修改 `Home.vue` 的播放按钮
- 使用用户选择的语音进行播放
- 显示当前使用的语音名称

### 6. 添加语音设置到翻译结果卡片
- 在翻译结果卡片显示当前语音
- 提供快速切换语音的选项

## 技术要点
- 使用 `speechSynthesis.getVoices()` 获取可用语音
- 按语言代码分组（en-US/en-GB）
- 优先推荐常用语音（Google US English, Microsoft David, Microsoft Zira等）
- 使用 localStorage 持久化用户设置
